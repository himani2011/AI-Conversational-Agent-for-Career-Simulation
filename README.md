An AI-driven simulation project demonstrating how LLMs can power interactive and adaptive career guidance systems.

	•	Developed an AI-powered conversational agent to simulate real-world career coaching and role-based interactions
	•	Built using LLMs with LangChain to enable context-aware, dynamic conversations
	•	Implemented Retrieval-Augmented Generation (RAG) with a custom knowledge base for accurate and personalized responses
	•	Designed and integrated REST APIs (Flask) to connect the AI system with front-end applications
	•	Structured data storage using relational databases to capture user interactions and support analytics
	•	Focused on low-latency responses and scalable architecture for real-time user engagement
	•	Explored use cases in interactive learning, simulation-based training, and intelligent virtual assistants
